## About Me

I'm a passionate and self-taught programmer with a knack for building robust and scalable web applications. My journey in development has been enriched by hands-on experience across various roles, leveraging open-source resources to hone my skills.

Currently, I'm a **Full Stack Developer at Slashash Tech LLP in Jodhpur**, where I specialize in managing relational databases like MySQL and PostgreSQL. My contributions primarily revolve around SaaS and AI-based projects, alongside developing tools such as a DNS resolver using JavaScript libraries. This role has significantly deepened my expertise in both frontend and backend development.

Prior to this, I interned as a **Frontend Developer at Sharma-store** in Jaipur, where I played a key role in designing and developing an e-commerce site's frontend functionalities using Next.js and Material UI. I focused on creating user-friendly interfaces to ensure a seamless shopping experience for customers.

I hold a **Bachelor of Computer Application (BCA)** from St. Wilfred's PG College, Jaipur.
